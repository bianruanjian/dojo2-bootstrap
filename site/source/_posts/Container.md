# Container

1. 名称：dojo-Container
1. 是否容器：true

## 属性

| 名称  | 说明 | 默认值 | 可选值 |值类型 |
| ----- | ------ | ----- | ----- | --------- |
| fluidWidth | 水平铺满 | false |          | boolean |

## 示例

固定宽度

<dojo-Container>水平居中</dojo-Container>

```html
<dojo-Container>
水平居中
</dojo-Container>
```

铺满全屏

```html
<dojo-Container fluidWidth="true">
铺满全屏
</dojo-Container>
```
