  2. 在部件的设计中发现以下属性都在描述“是否水平铺满显示”，需考虑能否统一名称：
    1. Container 的 fluidWidth -> Container 的 fluid
    2. Button 的 block -> Button 的 fluid
    3. Checkbox 的 inline -> Checkbox 的 fluid
    4. Radio 的 inline -> Radio 的 fluid
    5. Image 的 fluid -> Image 的 fluid
